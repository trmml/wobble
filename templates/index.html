Wobble

Wobble is a real-time native earthquake event manager that provides nearly instant significant earthquake updates and information regarding them. It comes neatly wrapped with a nice API, to boot.

About 6 months ago, @WAGGNews asked me to make a Twitter bot for them that tweets real-time earthquake events and the information regarding them. I did, sort of. Although the functionality was there, I never finished connecting it to Twitter or pushing it to a server.

I want to finish that project, though, because I enjoyed working on it, and I'd like to help out WAGG as much as possible. But I'd also like to create a service around this functionality, because I think it'd serve as a helpful tool to anyone who wishes to monitor recent earthquakes, for whatever reason that may be.

I'll include a link here when I finish with the Twitter bot (hopefully within this week.)

API Endpoint

GET /api/recent

Provides a sentence of the most recent significant earthquake. This includes location, magnitude, and time. Also provides the appropriate usgs.gov event URL.
